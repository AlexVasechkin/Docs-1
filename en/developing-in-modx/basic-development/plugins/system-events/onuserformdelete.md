## Event: OnUserFormDelete

Fires after a User is deleted.

Service: 1 - Parser Service Events 
Group: modUser

## Event Parameters

| Name | Description |
|------|-------------|
| user | A reference to the modUser object. |
| id | The ID of the user. |
