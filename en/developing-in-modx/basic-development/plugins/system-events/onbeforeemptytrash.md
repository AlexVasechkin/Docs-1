## Event: OnBeforeEmptyTrash

Fires before the trash is emptied for the site.

Service: 1 - Parser Service Events 
Group: Documents

## Event Parameters

| Name | Description |
|------|-------------|
| ids | An array of Resource IDs that will be permanently deleted. |
