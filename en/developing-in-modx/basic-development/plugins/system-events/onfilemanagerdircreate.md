Event: OnFileManagerDirCreate
-----------------------------

 Fires after creating a new directory in the manager.

 Service: 1 - Parser Service Events   
 Group: None

Event Parameters
----------------

 <table><tbody><tr><th> Name </th> <th> Description </th> </tr><tr><td> directory </td> <td> Contains the full path of the newly created directory. </td> </tr><tr><td> source </td> <td> Contains the mediasource object. </td></tr></tbody></table>See Also
