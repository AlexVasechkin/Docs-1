## Event: OnBeforeSnipFormDelete

Fires before a Snippet is deleted in the manager.

Service: 1 - Parser Service Events 
Group: Snippets

## Event Parameters

| Name    | Description                           |
| ------- | ------------------------------------- |
| snippet | A reference to the modSnippet object. |
| id      | The ID of the Snippet.                |
