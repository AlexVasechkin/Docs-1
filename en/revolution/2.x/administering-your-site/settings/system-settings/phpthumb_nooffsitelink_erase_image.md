phpthumb\_nooffsitelink\_erase\_image
-------------------------------------

**Name**: phpThumb Offsite Linking Erase Image   
**Type**: Yes/No   
**Default**: Yes   
**Available In:** 2.0.8+

Indicates if an image linked from a remote server should be erased when not allowed.
