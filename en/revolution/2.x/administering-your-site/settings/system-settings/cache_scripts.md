cache\_scripts
--------------

**Name**: Enable Script Cache   
**Type**: Yes/No   
**Default**: Yes

When enabled, MODx will cache all Scripts (Snippets and Plugins) to file to reduce load times. MODx recommends leaving this set to 'Yes'.
