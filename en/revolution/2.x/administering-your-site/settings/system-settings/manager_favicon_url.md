manager\_favicon\_url
---------------------

**Name**: Manager Favicon URL   
**Type**: Textfield   
**Default**:   
**Available In:** 2.1+

If set, will load this URL as a favicon for the MODX manager. Must be a relative URL to the manager/ directory, or an absolute URL.
