## symlink\_merge\_fields

**Name**: Merge Resource Fields in Symlinks 
**Type**: Yes/No 
**Default**: Yes 
**Available In:** 2.0.8+

If set to Yes, will automatically merge non-empty fields with target resource when forwarding using Symlinks.
