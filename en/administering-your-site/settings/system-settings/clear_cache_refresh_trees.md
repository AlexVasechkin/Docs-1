clear\_cache\_refresh\_trees
----------------------------

**Name**: Refresh Trees on Cache Clear   
**Type**: Yes/No   
**Default**: No   
**Available In**: 2.0.7+

If set to 'yes', this will refresh all active trees when the cache is cleared in the manager.
