## xPDO::setDebug

Sets the debug state for the XPDO connection.

## Syntax

API Docs: <http://api.modxcms.com/xpdo/xPDO.html#setDebug>

``` php 
void setDebug ([boolean $v = true])
```

## Example

``` php 
$xpdo->setDebug(true);
```
