## Event: OnBeforeChunkFormDelete

Fires before a chunk is deleted.

Service: 1 - Parser Service Events
Group: Chunks

## Event Parameters

| Name  | Description                         |
| ----- | ----------------------------------- |
| chunk | A reference to the modChunk object. |
| id    | The ID of the Chunk.                |
