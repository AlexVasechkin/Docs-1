## Event: OnPluginSave

Happens right after a plugin is removed.

Service: 2 - Manager Access Events 
Group: modPlugin

## Event Parameters

| Name   | Description                          |
| ------ | ------------------------------------ |
| plugin | A reference to the modPlugin object. |
