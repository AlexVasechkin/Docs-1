## phpthumb\_nohotlink\_erase\_image

**Name**: phpThumb Hotlinking Erase Image
**Type**: Yes/No
**Default**: Yes
**Available In:** 2.0.8+

Indicates if an image generated from a remote server should be erased when not allowed.
