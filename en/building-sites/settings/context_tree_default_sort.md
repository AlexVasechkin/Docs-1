## context\_tree\_sort

 **Name**: Enable Sorting of Contexts in Resource Tree
**Type**: Yes/No
**Default**: No
**Available In:** 2.1.2+

 If the context(s) in the resource tree should be sorted by the system settings context\_tree\_sortby and context\_tree\_sortdir
