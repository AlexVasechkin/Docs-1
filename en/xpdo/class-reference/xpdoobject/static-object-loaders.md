2. [\_loadInstance](xpdo/class-reference/xpdoobject/static-object-loaders/loadinstance)
3. [\_loadCollectionInstance](xpdo/class-reference/xpdoobject/static-object-loaders/loadcollectioninstance)
4. [load](xpdo/class-reference/xpdoobject/static-object-loaders/load)
5. [loadCollection](xpdo/class-reference/xpdoobject/static-object-loaders/loadcollection)
6. [loadCollectionGraph](xpdo/class-reference/xpdoobject/static-object-loaders/loadcollectiongraph)
