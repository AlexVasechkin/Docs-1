## site\_start

**Имя**: Site Start
**Тип**: Number
**По умолчанию**: 1

Идентификатор ресурса, который вы хотите использовать в качестве домашней страницы.

Убедитесь, что введенный вами идентификатор принадлежит существующему опубликованному ресурсу!
