## context\_tree\_sort

**Имя**: Enable Sorting of Contexts in Resource Tree
**Тип**: Да/Нет
**По умолчанию**: Нет
**Доступно **: 2.1.2+

Если контекст(ы) в дереве ресурсов должны быть отсортированы по системным параметрам context\_tree\_sortby и context\_tree\_sortdir
