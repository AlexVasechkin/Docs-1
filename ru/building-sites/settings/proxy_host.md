## proxy\_host

**Имя**: Proxy Host
**Тип**: String
**По умолчанию**:

Имя хоста вашего прокси-сервера. Если это указано, MODX будет пытаться использовать прокси при подключении к каналам RSS или [Управление пакетами](extending-modx/transport-packages "Управление пакетами").
