## phpthumb\_cache\_source\_enabled

**Имя**: phpThumb Cache Source Files
**Тип**: Да/Нет
**По умолчанию**: Нет

Следует ли кэшировать исходные файлы при их загрузке. Рекомендуется оставить Нет
