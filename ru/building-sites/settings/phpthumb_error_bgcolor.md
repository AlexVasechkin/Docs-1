## phpthumb\_error\_bgcolor

**Имя**: phpThumb Error Background Color
**Тип**: textfield
**По умолчанию**: CCCCFF
**Доступно в**: 2.0.8+

Шестнадцатеричное значение без знака #, указывающее цвет фона для вывода ошибки phpThumb.
