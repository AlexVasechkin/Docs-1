## phpthumb\_error\_fontsize

**Имя**: phpThumb Error Font Size
**Тип**: textfield
**По умолчанию**: 1
**Доступно в**: 2.0.8+

Значение em, указывающее размер шрифта, который будет использоваться для текста, появляющегося при выводе ошибки phpThumb.
