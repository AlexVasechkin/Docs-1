## phpthumb\_far

**Имя**: phpThumb Force Aspect Ratio
**Тип**: Да/Нет
**По умолчанию**: Нет

Значение по умолчанию для phpThumb при использовании в MODX. По умолчанию установлено значение C для принудительного соотношения сторон к центру.
