## phpthumb\_error\_textcolor

**Имя**: phpThumb Error Text Color
**Тип**: textfield
**По умолчанию**: FF0000
**Доступно в**: 2.0.8+

Шестнадцатеричное значение без знака #, указывающее цвет текста для вывода ошибки phpThumb.
