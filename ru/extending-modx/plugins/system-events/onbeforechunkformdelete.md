## Событие: OnBeforeChunkFormDelete

Запускается до удаления фрагмента.

Сервис: 1 - Парсер Сервис событий
Группа: Chunks

## Параметры события

| Имя   | Описание                   |
| ----- | -------------------------- |
| chunk | Ссылка на объект modChunk. |
| id    | Идентификатор чанка.       |
