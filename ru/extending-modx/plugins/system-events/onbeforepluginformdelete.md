## Событие: OnBeforePluginFormDelete

Запускается до удаления плагина в менеджере.

Служба: 1 - Parser Service Events
Группа: Plugins

## Параметры события

| Имя    | Описание                    |
| ------ | --------------------------- |
| plugin | Ссылка на объект modPlugin. |
| id     | Идентификатор плагина.      |
