## Событие: OnBeforeEmptyTrash

Запускается до того как корзина для сайта очищается до мусора.

Служба: 1 - Parser Service Events
Группа: Documents

## Параметры события

| Имя | Описание                                            |
| --- | --------------------------------------------------- |
| ids | Массив ID ресурсов, которые будут навсегда удалены. |
