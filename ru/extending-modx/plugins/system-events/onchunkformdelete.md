## Событие: OnChunkFormDelete

Запускается после удаления чанка.

Служба: 1 - Parser Service Events
Группа: Chunks

## Параметры события

| Имя   | Описание                   |
| ----- | -------------------------- |
| chunk | Ссылка на объект modChunk. |
| id    | Идентификатор Чанка.       |
