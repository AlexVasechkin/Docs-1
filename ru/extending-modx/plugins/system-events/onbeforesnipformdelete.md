## Событие: OnBeforeSnipFormDelete

Запускается до удаления сниппета в менеджере.

Служба: 1 - Parser Service Events
Group: Snippets

## Параметры события

| Имя     | Описание                     |
| ------- | ---------------------------- |
| snippet | Ссылка на объект modSnippet. |
| id      | Идентификатор сниппета.      |
